Intentzio onenarekin, jakina

Ikastaro honetan zerotik abiatuko gara, ez da aldez aurretik programazioaz ezer jakin behar, ikasteko gogoa besterik ez da behar. Hauexek dira ikastaroaren lerro nagusiak:

  • HTML5, CSS3 eta JavaScript irakastea, haien xehetasunak aztertuz, ikasiz eta erabiliz.
  • Zalantzak argitzeko eta ikasitakoa garatzeko tokiak eta helbideak non diren erakustea.
  • Mundu errealeko profesionalek nola lan egiten duten erakustea, haien tresnak aurkeztuz, hala nola Github, Kode editoreak, nabigatzaileen xehetasunak edota zerbitzari lokalak.

Programazioaren munduan arazo bera konpontzeko modu asko daude, guztiak balekoak, emaitza zuzena bada. Halere, ikastaroaren helburua ez da problema bat ebazteko modu guzti-guztiak ikastea (hori urteetako praktikarekin ikasten da).

Gida didaktikoa

Glosarioa

Atal hau aurkibide orokor bat izango da. Bertan ikastaroan zehar erabilitako terminoak zerrendatuko dira bakoitza bere azalpenarekin eta aurkibide bizkor bategaz.

Material osagarriak

Web-gune-gintza osatzen, webgintza ikasten eta webguneak garatzen lagundu didaten webgune, liburu eta ikus-entzunezkoak zerrendatu nahi nituzke hemen.

Ikasmateriala osatzen duten ikasgaiak

1.- Web garapena

Internet zer den ikusita, web-orriak sortzeko eta webguneak osatzeko beharko ditugun elementuak eta lanabesak ikusiko ditugu hiru ataletan: WEBGUNEAK, LENGOAIAK eta LANABESAK.

2.- HTML5

Web-orrien edukia eta egitura, irudiak, zerrendak, bideoak eta abar definitzeko erabiltzen den HTML lengoaiaren xehetasunak ikasiko ditugu hiru ataletan: EGITURA, EDUKIA eta FORMULARIOAK.

3.- CSS Oinarrizkoak

CSSek eskaintzen dizkigun erregelak medio (hautatzailea + propietateak + balioa) HTML5az eraikitako web-orriei itxura emango diegu.

4.- CSS Diseinua

Web-orrietako elementuak web-orrian antolatzeko CSSk bi tresna eskaintzen digu: flex eta grid. Eta web-orriak pantaila tamaina ezberdinei egokitzeko media query izeneko funtzioa daukagu.

5.- JavaScript

JavaScriptek jarraibideak modu sekuentzialean exekutatzen ditu, sistemari adierazteko zer nahi dugun egin.
