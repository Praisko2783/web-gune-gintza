Heziketa-zikloa osatzen duten lanbide-moduluak:

  • Markatzeko lengoaiak eta informazioa kudeatzeko sistemak.
  • Garapen-inguruneak.
  • Web-garapena bezero-ingurunean.
  • Web interfazeak diseinatzea.
  • Datu-baseak.
  • Programazioa.
  • Web aplikazioen hedapena.
  • Sistema informatikoak.
  • Web-garapena zerbitzari-ingurunean.
  • Web aplikazioak garatzeko proiektua.
  • Ingeles teknikoa.
  • Laneko prestakuntza eta orientabidea.
  • Enpresa eta ekimen sortzailea.
  • Lantokiko prestakuntza.

Urdinez eta azpimarraz zehaztu ditut proiektuaren lehen joaldi honetan zuzen-zuzenean ukitu asmo ditudan moduluak. Egia da beste moduluetako batzuk (urdinez, azpimarra gabe) zeharka bada ere, aipatu aipatuko direla, eta askotan landu ere bai, hala nola Datu Baseak, Programazioa, Web aplikazioen hedapena..., baina ez dira izango proiektuaren zati honen helburu.

Zeharkako horiek 'Web-garapena zerbitzari-ingurunean' izenaz bataiatu asmo dudan proiektuaren bigarren joaldian landuko dira zuzen-zuzenean, eta honako honetan zuzen-zuzenean jorratuko direnak izango dira orduan, zeharka landuko direnak.
