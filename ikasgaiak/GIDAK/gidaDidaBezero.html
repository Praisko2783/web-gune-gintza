Web garapena bezeroaren ingurunean

Ikaskuntzaren emaitzak eta ebaluazio-irizpideak.

1.- Web-bezeroen gaineko programazio-teknologiak eta -arkitekturak hautatzen ditu, bakoitzaren gaitasunak eta ezaugarriak identifikatuta eta aztertuta.

Ebaluazio-irizpideak:
  1. Zerbitzarian eta web bezeroan kodea exekutatzeko ereduak ezaugarritu eta bereizi ditu.
  2. Web nabigatzaileen gaitasunak eta kodea exekutatzeko mekanismoak identifikatu ditu.
  3. Web bezeroen programazioarekin lotzen diren lengoaia nagusiak identifikatu eta ezaugarritu ditu.
  4. Gidoien programazioaren berezitasunak ezagutu ditu, bai eta programazio tradizionalaren gainean dituen abantailak eta desabantailak ere.
  5. Markatzeko lengoaiak web bezeroen programazio-lengoaiekin integratzeko mekanismoak egiaztatu ditu.
  6. Web bezeroen gaineko programazio-tresnak ezagutu eta ebaluatu ditu.

2.- Epai sinpleak idazten ditu, hizkuntzaren sintaxia aplikatuta eta web nabigatzaileen gainean exekutatzen dela egiaztatuta

Ebaluazio-irizpideak:
  1. Web bezeroak programatzeko lengoaia hautatu du, aukeren arabera.
  2. Hizkuntzan erabilgarri dauden aldagai eta eragile motak erabili ditu.
  3. Aldagaien erabilera-esparruak identifikatu ditu.
  4. Hizkuntzaren berezitasunak ezagutu eta egiaztatu ditu, datu moten arteko bihurketei dagokienez.
  5. Epaien blokeak sortzeko erabaki-mekanismoak erabili ditu.
  6. Kodeari iruzkinak gehitu dizkio.
  7. Begiztak erabili ditu eta funtzionamendua egiaztatu du.
  8. Kodearen programazioa, proba eta arazketa errazteko tresnak eta inguruneak erabili ditu.

3.- Kodea idazten du, eta, horretarako, lengoaiaren objektu aurredefinituek ekarritako funtzionaltasunak identifikatzen eta aplikatzen ditu.

Ebaluazio-irizpideak:
  1. Lengoaiaren objektu aurredefinituak identifikatu ditu.
  2. Nabigatzailearen leihoei dagozkien objektuak eta horien web-dokumentuak aztertu ditu.
  3. Lengoaiaren objektu aurredefinituak nabigatzailearen itxura eta jasotzen duen dokumentua aldatzeko erabiltzen dituzten sententziak idatzi ditu.
  4. Nabigatzailean kodea gauzatzearen ondorioz testuak eta etiketak sortu ditu.
  5. Lengoaiaren objektu aurredefinituak erabiltzailearekin elkarreragiteko erabiltzen dituzten sententziak idatzi ditu.
  6. Hainbat leihoz eta markoz osatutako dokumentuetan hizkuntzaren berezko ezaugarriak erabili ditu.
  7. Cookieak erabili ditu informazioa biltegiratzeko eta edukia berreskuratzeko. h) Kodea araztu eta dokumentatu du.

4.- Web bezeroentzako kodea programatzen du, erabiltzaileak definitutako egiturak aztertuz eta erabiliz.

Ebaluazio-irizpideak
  1. Lengoaiaren funtzio aurredefinituak sailkatu eta erabili ditu.
  2. Erabiltzaileak definitutako funtzioak sortu eta erabili ditu.
  3. Arrayak sortzeari eta erabiltzeari lotutako lengoaiaren ezaugarriak ezagutu ditu.
  4. Array-ak sortu eta erabili ditu.
  5. Hizkuntzaren objektuetara orientatzeko ezaugarriak ezagutu ditu.
  6. Objektuen egitura definitzeko kodea sortu du.
  7. Metodoak eta propietateak sortu ditu.
  8. Erabiltzaileak definitutako objektuak erabiltzen dituen kodea sortu du.
  9. Kodea araztu eta dokumentatu du.

5.- Web aplikazio elkarreragileak garatzen ditu, gertaerak maneiatzeko mekanismoak integratuta.

Ebaluazio-irizpideak
  1. Sortutako gertaerak atzitzeari dagokionez, markatzeko lengoaiaren aukerak ezagutu ditu.
  2. Ekitaldien kudeaketari buruzko programazio-lengoaiaren ezaugarriak identifikatu ditu.
  3. Maneiatu daitezkeen gertaera motak bereizi ditu.
  4. Gertaerak harrapatu eta erabiliko dituen kodea sortu du.
  5. Web inprimakiak kudeatzeko lengoaiaren gaitasunak ezagutu ditu.
  6. Web formularioak balidatu ditu ekitaldiak erabilita.
  7. Baliozkotze-prozedurak errazteko adierazpen erregularrak erabili ditu.
  8. Kodea probatu eta dokumentatu du.

6.- Web aplikazioak garatzen ditu, dokumentuko objektuen ereduaren ezaugarriak aztertu eta aplikatuta.

Ebaluazio-irizpideak
  1. Web orri bateko dokumentuaren objektuen eredua ezagutu du.
  2. Modeloaren objektuak, propietateak eta metodoak identifikatu ditu.
  3. Dokumentuaren egiturara sartzen den kode bat sortu eta egiaztatu du.
  4. Egituraren elementu berriak sortu ditu eta lehendik zeuden elementuak aldatu ditu.
  5. Ekintzak ereduko gertaerekin lotu ditu.
  6. Ereduak hainbat nabigatzailetan dituen desberdintasunak identifikatu ditu.
  7. Web aplikazioak programatu ditu, ereduaren hainbat inplementazio dituzten nabigatzaileetan funtzionatzeko moduan.
  8. Web aplikazioetan hiru alderdiak (edukia, itxura eta portaera) independizatu ditu.

7.- Web aplikazio dinamikoak garatzen ditu, eta, horretarako, bezeroaren eta zerbitzariaren arteko komunikazio asinkronoko mekanismoak ezagutzen eta aplikatzen ditu.

Ebaluazio-irizpideak
  1. Bezeroaren eta web zerbitzariaren arteko komunikazio asinkronoko mekanismoak erabiltzearen abantailak eta eragozpenak ebaluatu ditu.
  2. Komunikazio asinkronoa ezartzeko erabilgarri dauden mekanismoak aztertu ditu.
  3. Erlazionatutako objektuak erabili ditu.
  4. Propietateak eta metodoak identifikatu ditu.
  5. Komunikazio asinkronoa erabili du web-dokumentuaren eguneratze dinamikoan.
  6. Informazioa bidaltzeko eta jasotzeko hainbat formatu erabili ditu.
  7. Web aplikazio asinkronoak programatu ditu, hainbat nabigatzailetan funtzionatzeko moduan.
  8. Web-orrien programazioan eguneratze dinamikoko teknologiak sartzea erraztuko duten liburutegiak sailkatu eta aztertu ditu.
  9. Liburutegi horiek erabiltzen dituzten programak sortu eta araztu ditu.
