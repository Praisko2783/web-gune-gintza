Ebaluazio-irizpideak

  • Garatu beharreko web aplikazioen interfazerako estilo-gida eta diseinua interpretatu ditu, diseinu-ekipoaren jarraibideak kontuan izanda.
  • Web aplikazioen interfazea osorik edo partzialki osatzen duten markatzeko lengoaietan sententzia-blokeak editatu eta probatu ditu, estiloak kanpo-orrietatik administratuz.
  • Informazioa bezeroaren aldean aurkezpen-formatu egokietara bihurtzeko eta egokitzeko transformazio-orriak erabili ditu.
  • Web aplikazio baten interfazerako multimedia-materialak prestatzen eta integratzen parte hartu du, diseinu-ekipoaren jarraibideei jarraituz.
  • Ekitaldiak maneiatzean eta elementu interaktiboak eta animazioak integratzean oinarritutako web aplikazio interaktiboak garatzen lagundu du.
  • Web aplikazioen erabilerraztasuna eta erabilgarritasuna egiaztatu ditu, eta eskatutako mailak betetzeko beharrezko aldaketak eta neurriak egiten lagundu du.
  • Web-aplikazioen zerbitzari-zatia diseinatzeko eta garatzeko arduradunekin lankidetza estuan aritu da, irizpideak bateratuz eta aplikazioaren bi aldeetako garapena koordinatuz.
  • Aplikazioaren erabiltzaileari laguntzeko protokoloetan erabilitako dokumentazioa eta gainerako osagaiak definitzen eta lantzen parte hartu du.
