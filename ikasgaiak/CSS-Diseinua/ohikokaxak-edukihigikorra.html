Ohiko kaxetan, eduki higikorrak

Higikortasuna

Aipatu dugun bezala, Block elementuak elkarren gainean jartzen dira, eta Inline elementuak ezkerretik eskuinera kokatzen dira lerro berean. Kutxa tradizionalaren ereduak ezartzen duenez, elementuek leihoaren alde banatan flotatu dezakete, eta lerro berean partekatu dezakete espazioa beste elementu batzuekin, edozein motatakoak direla ere. Adibidez, diseinuan zutabeak irudikatzen dituzten bi Block elementu baditugu, zutabe bat ezkerrean eta bestea eskuinean koka ditzakegu, elementuak nahi dugun aldera flotaraziz. Honako hauek dira CSSk helburu horretarako eskaintzen dituen propietateak.
