Media Query

Media Query bat CSSn erreserbatutako arau bat da, garatzaileei dokumentua erakusten den ingurunea detektatzeko aukera emateko asmoz txertatu zena. Adibidez, Media Queries erabiliz, dokumentua monitore batean erakusten den edo inprimagailu batera bidaltzen den zehaztu dezakegu, eta kasu bakoitzerako estilo egokiak esleitu. Horretarako, Media Querie-ek gako-hitz hauek eskaintzen dituzte.
