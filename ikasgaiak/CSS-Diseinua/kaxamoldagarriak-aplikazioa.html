Egitura-elementuen antolaketa bera denez, dokumentuari aplikatu behar dizkiogun CSS arauak ere antzekoak dira. Egiturazko elementuak edukiontzi malgu bihurtu besterik ez dugu egin behar, eta diseinuak hala eskatzen duenean haien edukia malgu bihurtu. Adibidez, hurrengo arauek <header> elementuaren display propietateari flex modua esleitzen diote, eta elementu horren barruko <div> elementua malgu gisa deklaratzen dute, goiburua eta haren edukia zabaldu egin daitezen, leihoan dagoen espazio osoa okupatu arte. 

        
* {
  margin: 0 auto;
  padding: 0
}

header {
  width: 100%;
  height: 150px;
  padding: 0% 2%;
  background-color: #0F76A0;
  margin: 0px auto;
}

header h1 {
  font: bold 54px Arial, sans-serif;
  color: #FFFFFF;
  text-align: center;
}

Goiburuko edukia pantailan zentratuta geratzea nahi dugunez, center balioa justify-content propietateari. <div> elementua malgutzat jo da, eta horrek esan nahi du espazio erabilgarriaren arabera hedatu edo murriztuko dela, baina, lehen azaldu dugun bezala, orria pantaila zabaleko gailuetan aurkezten denean, ziurtatu behar dugu ez dela zabalegia eta deserosoa erabiltzailearentzat. Hori max-width propietatearekin konpontzen da. Propietate horri esker, <div> elementua ez da 960 pixel baino gehiago hedatuko.

Eduki nagusirako arauak ere berberak dira, baina oraingoan bi edukiontzi malgu ditugu: bata, edukia orrian zentratzeko ardura duen <main> elementua, eta bestea, <section> eta <aside> elementuek sortutako bi zutabeak konfiguratzeko ardura duen <div> elementua. Horregatik, <div> elementuari eragiten dion erregelak bi propietate hauek behar ditu: display, elementua edukiontzi malgu gisa deklaratzeko, eta flex, edukiontzia bera elementu malgu gisa deklaratzeko.

  flex-wrap: wrap;
  width: 70%;
  padding: 2%;
}

section {

  width: 60%;
  padding: 10px;
  background-color: #FFFFFF;
  border-radius: 10px;

Egin ezazu zeuk:

Sortu HTML fitxategi berri bat eta bertan goian OHIKO KUTXEN EREDUA izenarekin kodea itsatsi. Ondoren nirestiloa.css izeneko fitxategian orri honetako koadroetan dauzkazun kode zatiak bata bestearen azpian itsatsi. Gogoratu nireirudia.jpg fitxategia direktorio berean sartzea. Ireki dokumentua zure nabigatzailean. Kutxa tradizionalaren ereduarekin aurretik sortutako orri bera ikusi beharko zenuke. Baina oraingoan orriko atalak malguak direnez (leihoaren tamaina handitu edo murriztu ahala artikuluen tamainak aldatu egingo dira) eta <aside> elementuak sortutako zutabea eremu nagusiaren azpian jarriko da.

Frogak egiteko 'frogalekua'

Estekak froga simpleak egiteko aproposa den aplikazio bat irekiko dizu elementuek zer-nolako itxura hartzen duten ikusteko, baina hor egindako frogak komeni da kode editorera eramatea eta bertan egiaztatzea gure nahiari hurbiltzen zaion egindakoa.
