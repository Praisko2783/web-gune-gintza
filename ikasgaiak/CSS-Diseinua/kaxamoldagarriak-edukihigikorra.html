Kaxa malguak

Edukiontzi malguak

Kutxa-eredu baten helburu nagusia mekanismo bat eskaintzea da, leihoan dagoen espazioa zatitzeko eta web-orri baten diseinuaren zati diren errenkadak eta zutabeak sortzeko. Hala ere, kutxa-eredu tradizionalak eskaintzen dituen tresnek ez dute helburu hori betetzen. Kutxak nola banatu eta horien tamainak zehaztea, adibidez, ezin da modu eraginkorrean lortu eredu horrekin.

Diseinu-patroi komunak ezartzeko zailtasunak, hala nola, zutabeak zabaltzea erabilgarri dagoen espazioa okupatzeko, kaxa baten edukia ardatz bertikalean zentratzea edo zutabe bat orrialde baten goialdetik behealdera zabaltzea, edukiaren tamaina gorabehera, beste aukera batzuk bilatzera behartu du industria. Hainbat kutxa-eredu garatu dira, baina batek ere ez du kutxa malguaren ereduak baino arreta handiagoa jaso.

GARRANTZITSUA

Nahiz eta kutxa-eredu malguak bere abantailak dituen kutxa-eredu tradizionalarekiko, oraindik egoera esperimentalean dago eta nabigatzaile batzuek ezin dituzte bere propietateak prozesatu. Horregatik, kapitulu honetan kutxa tradizionalaren eredua ere sartu dugu. Zure webguneetan aplikatu beharreko eredua aukeratu aurretik, kontuan hartu behar dituzu muga horiek. HTML5ek sartutako edozein tresna erabil dezakezun jakiteko, bisitatu www.caniuse.com.

Edukiontzi malgua

Kutxa-eredu malguak modu dotorean konpontzen ditu kutxa-eredu tradizionalaren arazoak. Eredu honek kutxa tradizionalaren ereduak erabiltzen dituen tresnak aprobetxatzen ditu, hala nola posizionamendu absolutua eta zutabeak, baina elementuak flotarazi beharrean kutxak edukiontzi malguak erabiliz antolatzen ditu. Edukiontzi malgua edukia kaxa malgu bihurtzen duen elementu bat da. Eredu berri honetan, kutxa-multzo bakoitzak beste kutxa baten barruan egon behar du, eta kutxa horrek bere ezaugarriak, hurrengo adibideak erakusten duen bezala.

          
  <!DOCTYPE html>
    <html lang="es">
      <head>
        <title>Hmen doa orriaren izenburua</title>
        <meta charset="utf-8">
        <meta name="description" content="Hau HTML5 dokumentua da">
        <meta name="keywords" content="HTML, CSS, JavaScript">
        <link rel="stylesheet" href="nirestiloa.css">
      </head>
    <body>
      <section id="kaxaNagusia">
          <div id="kaxa-1">Kaxa 1</div>
          <div id="kaxa-2">Kaxa 2</div>
          <div id="kaxa-3">Kaxa 3</div>
          <div id="kaxa-4">Kaxa 4</div>
      </section>
      </body>
    </html>

Goiko dokumentuak, aurreko adibideetan bezala, beste elementu batzuen edukiontzi gisa jarduten duen elementu bat du. <section> CSStik elementuak nola konfiguratzen diren, horixe da aldea. Kaxak elementuaren barruan malgu bihurtzeko (tamainak aldatu egiten dira, dagoen espazioaren arabera), elementu hori edukiontzi malgu bihurtu behar dugu. <section> Horretarako, CSSk display propietaterako flex eta inline-flex balioak eskaintzen ditu. flex balioak block elementu malgua definitzen du, eta inline-flex balioak inline elementu malgua.

          
            #kaxaNagusia {
              display: flex;
              }
Egizu zu zeuk

Sor ezazu HTML fitxategi berri bat Goiko HTML kodearekin, lotu nirestiloa.css fitxategiarekin eta, azken horretan idatzi goiko erregela. Ireki dokumentua zure nabigatzailean. kaxanagusia elementuaren barruan bata bestearen ondoan, <section> elementuaren barruan ikusi beharko zenituzke, lerrokatuta eta CSS erregelarik jantzi gabe, biluzik.

Frogak egiteko 'frogalekua'

Estekak froga simpleak egiteko aproposa den aplikazio bat irekiko dizu elementuek zer-nolako itxura hartzen duten ikusteko, baina hor egindako frogak komeni da kode editorera eramatea eta bertan egiaztatzea gure nahiari hurbiltzen zaion egindakoa.
