Zutabeak web diseinuan

Kaxak pantailan antolatzeko aztertu ditugun propietateez gain, CSSk propietate multzo bat ere badu, zutabeak errazago sortzeko.

  • column-count — Propietate honek zehazten du nabigatzaileak zenbat zutabe sortu behar dituen elementuaren edukia banatzeko.
  • column-width —Propietate honek zutabeen zabalera adierazten du. Balioa auto gisa (lehenetsita) edo CSS unitateetako edozeinetan, pixel edo portzentaje gisa deklara daiteke.
  • column-span —Ezaugarri hau kutxaren barruko elementuei aplikatzen zaie, elementua zutabe batean edo zutabe batzuen artean banatuta kokatuko den zehazteko. Balio erabilgarriak all (zutabe guztiak) eta none (lehenetsita) dira.
  • column-fill —Propietate honek zehazten du edukia nola banatuko den zutabeen artean. Balio erabilgarriak autokoak (zutabeak sekuentzialki osatzen dira) eta balantzekoak dira (edukia zati berdinetan banatzen da zutabe guztien artean).
  • column-gap —Propietate honek zutabeen arteko espazioaren tamaina definitzen du. Onartu balio bat CSSn eskuragarri dagoen edozein unitatetan, pixeletan eta ehunekoetan, esaterako.
  • columns —Propietate honek aukera ematen digu columncount eta column-width propietateen balioak aldi berean deklaratzeko.

Edukia zutabetan banatu nahi dugun elementua elementu komuna da, eta edukia lehen egin dugun bezala adierazten da. Nabigatzailea zutabeak sortzeaz eta edukia guk zatitzeaz arduratzen da. Hurrengo adibideak < article> elementu bat du, bi zutabetan aurkeztuko dugun testu luze batekin.

        
          <!DOCTYPE html>
          <html lang="es">
          <head>
            <title>Hemen doa dokumnetuaren izena</title>
            <meta charset="utf-8">
            <meta name="description" content="Hau HTML5 dokumentua da">
            <meta name="keywords" content="HTML, CSS, JavaScript">
            <link rel="stylesheet" href="nirestiloa.css">
          </head>
          <body>
            <section>
              <article id="albisteArtikulua">
                <span>HTML, HyperText Markup Language siglaren ingelesezko siglak (hipertestu-marken lengoaia), web-orriak egiteko markatzeko lengoaiari egiten dio erreferentzia. Web-orrien bertsio desberdinekin konektatzen duen softwarearen erreferentzia gisa balio duen estandar bat da. Web-orri baten edukia definitzeko oinarrizko egitura eta kode bat (HTML kodea) definitzen ditu, hala nola testua, irudiak, bideoak eta jokoak, besteak beste. World Wide Web Consortium (W3C) edo WWW Partzuergoaren ardurapeko estandarra da, webguneari lotutako teknologia ia guztiak estandarizatzen dituen erakundea, batez ere idazketari eta interpretazioari dagokienez.</span>
              </article>
            </section>
          </body>
          </html>

Zutabeak sortzeko propietateak edukiontziari aplikatu behar zaizkio. Hurrengo erregelak kolunn-count propietatea barne hartzen du, < article> elementuaren edukia bi zutabetan zatitzeko.

        
          #albisteArtikulua {
            width: 580px;
            padding: 10px;
            border: 1px solid;
            column-count: 2;
            column-gap: 20px;
            }

Goiko erregelak 580 pixeleko tamaina ematen dio <article> elementuari, eta edukiontziak hartzen duen eremua ertz solido batekin identifikatzen du. Erregelak 10 pixeleko betegarria ere badu, testua ertzetik bereizteko. Gainerako propietateek edukia bi zutabetan banatzen dute, 20 pixeleko tartearekin. Emaitza 4-17 irudian ageri da.

  • Column-gap propietateak definitzen du zutabeen arteko espazioaren tamaina. Bereizketa hori espazio hutsa baino ez da, baina CSSk honako propietate hauek eskaintzen ditu erabiltzaileari zatiketa bistaratzen lagunduko dion lerro bat sortzeko.
  • column-rule-style —Propietate honek banaketa irudikatzeko erabiltzen den lerroaren estiloa definitzen du. Balio erabilgarriak hidden (lehenetsita), dotted, dashed, solid, double, groove, ridge, inset eta outset dira.
  • column-rule-color —Propietate honek zatiketa irudikatzeko erabiltzen den lerroaren kolorea zehazten du.
  • column-rule-width —Propietate honek zatiketa irudikatzeko erabiltzen den linearen zabalera zehazten du.
  • column-rule —Propietate honek aukera ematen digu linearen balio guztiak aldi berean definitzeko.

Zutabeen erdian lerro bat marrazteko, gutxienez bere estiloa zehaztu behar dugu. Hurrengo adibideak kolunn-rule propietatea inplementatzen du pixel bateko lerro beltz bat sortzeko.

        
        #albisteArtikulua {
        width: 580px;
        padding: 10px;
        border: 1px solid;
        column-count: 2;
        column-gap: 20px;
        column-rule: 1px solid # 000000;
        }
Egin ezazu zuk zeuk

Sortu HTML fitxategi berri bat goiko koadroko kodearekin. Hori eginda sortu nirestiloa.css izeneko CSS fitxategi bat goiko koadroan zutabeak zehazteko osatu dugun css kodearekin. Ireki dokumentua zure nabigatzailean. Testua bi zutabetan banatuta ikusi beharko zenuke.

Orain #albisteArtikulua erregela zutabeak banatzeko lerroa egiteko erregelarekin ordeztu eta eguneratu orria zure nabigatzailean. Zutabeak banatzen dituen lerro bat ikusi beharko nuke, goiko irudiak erakusten duen bezala.

Frogak egiteko 'frogalekua'

Estekak froga simpleak egiteko aproposa den aplikazio bat irekiko dizu elementuek zer-nolako itxura hartzen duten ikusteko, baina hor egindako frogak komeni da kode editorera eramatea eta bertan egiaztatzea gure nahiari hurbiltzen zaion egindakoa.
