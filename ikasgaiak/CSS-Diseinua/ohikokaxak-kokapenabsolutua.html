Posizio absolutua

Elementu baten posizioa erlatiboa edo absolutua izan daiteke. Posizio erlatibo batekin, kutxak bata bestearen ondoren jartzen dira edukiontziak adierazitako lekuan.

Espazioa nahikoa ez bada edo elementuak ez badira flotagarriak, kutxak lerro berri batean jartzen dira. Hori da lehenetsitako posizionamendu modua, baina badago beste modu bat, posizionamendu absolutua.

Posizionamendu absolutuak aukera ematen digu elementu bakoitza zein koordenatutan kokatu nahi dugun zehazteko. Honako hauek dira helburu horretarako eskura dauden propietateak.

  • position —Propietate honek elementu bat jartzeko erabiltzen den posizionamendu mota definitzen du. Honako hauek dira balio erabilgarriak: , .
    • static (dokumentuaren fluxu normalaren arabera kokatzen da)
    • relative (elementuaren jatorrizko posizioaren arabera kokatzen da)
    • absolute (elementuaren edukiontziari dagokion posizio absolutuarekin kokatzen da)
    • fixed (nabigatzailearen leihoari dagokion posizio absolutuarekin kokatzen da).
  • top —Propietate honek elementuaren goiko ertzaren eta edukiontziaren goiko ertzaren arteko distantzia zehazten du.
  • bottom —Propietate honek elementuaren beheko marjinaren eta edukiontziaren beheko marjinaren arteko distantzia zehazten du.
  • left —Propietate honek elementuaren ezkerreko marjinaren eta edukiontziaren ezkerreko marjinaren arteko distantzia zehazten du.
  • right —Propietate honek elementuaren eskuineko marjinaren eta edukiontziaren eskuineko marjinaren arteko distantzia zehazten du.

top, bottom, left, eta right propietateak bi posizionamendu motetan aplikatzen dira, erlatiboa edo absolutua, baina modu desberdinetan lan egiten dute. Elementua kokapen erlatiboarekin kokatzen denean, elementua mugitu egiten da diseinua aldatu gabe. Posizioa absolutua bada, elementua diseinutik ezabatzen da, beraz, elementuak lekuz aldatzen dira espazio libre berria okupatzeko.

Hurrengo adibideak posizionamendu erlatiboa erabiltzen du gure adibideko lehen kutxa 25 pixel beherantz mugitzeko.

        
          #kaxa-1 {
            position: relative;
            top: 25px;
            float: left;
            width: 140px;
            height: 50px;
            margin: 5px;
            background-color: #AAAAAA;
        }
Egin ezazu zuk zeuk

ordezka ezazu 1 kaxa-erregela zure CSS estilo-orrian, goiko erregelarekin. Adibide honetan, lehenengo kutxa bakarrik desplazatzen dugu, baina ez ditugu arauak ukitzen gainerako kutxetarako. Gehitu beste propietate batzuk, hala nola left edo right, elementuaren posizioari nola eragiten dioten ikusteko.

Beste aukera bat posizio absolutua erabiltzea da. Kasu honetan, elementua diseinutik kentzen da, eta, beraz, gainerako elementuei eragiten die erregelak. Posizionamendu absolutua erabiltzen dugunean, kontuan hartu behar dugu elementua nabigatzailearen leihoarekiko kokatuko dela, elementu aitaren posizioa adierazten ez badugu behintzat. Beraz, aitaren posizioan oinarritutako elementu batentzako posizio absolutu bat zehaztu nahi badugu, aitarentzako jabetza positiboa ere aitortu behar dugu. Hurrengo adibidean, kaxapadre elementuarentzako posizio erlatibo bat eta 1 kaxaren goiko aldetik 25 pixeleko posizio absolutu bat adierazten ditugu. Egin ezazu zuk zeuk

Ordeztu #kaxanagusia eta #kaxa-1 erregelak zure CSS estilo-orrian, goiko koadroko erregelekin, eta ireki dokumentua zure nabigatzailean. Goiko Irudiaren antzeko zerbait ikusi beharko nuke.

Kodearen elementuen ordenak kutxak lapaginan kokatzeaz gain, gainjartzen direnean zein kutxa egongo den besteen gainetik ere zehazten du. Gure adibidean kaxa-1 kodean lehenengo deklaratu denez, kaxa-2aren gainean marrazten da, baina CSSk honako propietate hau eskaintzen du portaera hori aldatzeko.

  • z-index —Propietate honek elementuak z ardatzean duen posizioa zehazten duen indizea definitzen du. Indize altuena duen elementua indize txikiena duen elementuaren gainean marraztuko da.

Adibidez, 1. kutxa-elementua 2. kaxa-elementuaren azpian eta kutxa-elementuaren gainean mugi dezakegu, 1. kutxari eta kutxa-apadreari indize negatiboak esleituz.

        
          #kaxaNagusia {
            position: relative;
            width: 600px;
            border: 1px solid;
            z-index: -2;
            }
            #kaxa-1 {
            position: absolute;
            top: 25px;
            z-index: -1;
            float: left;
            width: 140px;
            height: 50px;
            margin: 5px;
            background-color: #AAAAAA;
            }

Indize negatiboak lehenetsitako indizeak baino baxuagotzat jotzen dira. Goiko kodean ikus dezakezuen bezala, z-index: -1; hau da -1 balioa esleitu diogu kutxa-1 elementuaren z-index propietateari, eta -2 balioa #kaxanagusia elementuaren z-index propietateari. Horrek kaxa-1 mugitzen du kaxa-2ren azpira, baina kaxa-1 kaxanagusiaren gainean mantentzen du, bere z-index propietatearen balioa altuagoa delako.
