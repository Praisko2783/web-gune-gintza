Gehien erabiltzen direnak none (elementua ezabatzen du), block (elementua lerro berri batean eta tamaina pertsonalizatuarekin erakusten du), inline (elementua lerro berean erakusten du) eta inlineblock (elementua lerro berean eta tamaina pertsonalizatuarekin erakusten du) dira.
Egiturazko elementuak block balioarekin konfiguratzen dira lehenetsita, eta edukia irudikatzen duten elementuak, berriz, inline gisa konfiguratu ohi dira. Elementu mota aldatu nahi badugu, display propietatea balio berri batekin esleitu besterik ez dugu egin behar. Horrela, nabigatzaileen bertsio zaharrek ez dituzte HTML5ek gehitutako elementu berriak ezagutzen, eta lehenetsitako Inline elementutzat hartzen dituzte. Elementu horiek nabigatzaile guztietan Block elementu gisa interpretatzen direla ziurtatu nahi badugu, hurrengo araua adieraz dezakegu gure estilo-orrietan.
