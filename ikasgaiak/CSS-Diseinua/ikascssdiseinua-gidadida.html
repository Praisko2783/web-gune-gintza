Atalaren gida didaktikoa: Web Diseinua (CSS)

CSSri eskainitako bigarren saio honetan Flexbox eta Grid moduluak erabiliko ditugu, zure proiektuetan pantailako elementuen posizionamendua modu sinpleagoan estilizatzeko. Modulu osoak dira, eta ez propietate bakarra; propietate multzo oso bat eskatzen dute.

Flexbox

Edukiontzi batean espazioa diseinatzeko, lerrokatzeko eta elementuen artean banatzeko modu eraginkorragoa ematea du helburu, baita haren tamaina ezezaguna eta/edo dinamikoa denean ere.

Diseinu malguaren atzean dagoen ideia nagusia edukiontziari bere elementuen zabalera/altuera (eta ordena) aldatzeko gaitasuna ematea da, erabilgarri dagoen espazioa hobeto betetzeko (pantaila-tamaina guztietara egokitzea). Edukiontzi malgu batek erabilgarri dagoen espazio librea betetzeko elementuak zabal ditzake, edo murriztu egin ditzake, gainezka ez egiteko.

  • CSS propietate hauek erabiliko dituzu "item" bat edo batzuk "edukiontzia" elementu baten barruan zentratzeko, horrela. 
        
        .edukiontzia {
            display: flex;
            justify-content: center;
            align-items: center;
            }
  • "edukiontzia" elementuaren noranzkoa aldatuko duzu, honako propietate hauekin. 
        
        .edukiontzia {
            display: flex;
            flex-direction: column;
        }
  • "edukiontzia" elementuaren barruko "item" elementuak tamainan doitzea nahi baduzu, hura handitzen edo koxkatzen denean, honako propietate hauek dituzu. 
        
        .item {
            display: flex;
            flex-grow: 1;
            flex-shrink: 1;
            flex-basis: auto;
        
            /* Modu laburrean gauza bera */
            flex: 1 1 auto;
        }
  • Normalean padding eta margin erabili ohi ditugu elementuen artean tarteak ipintzeko, baina horien erabilerak tarte espezifiko bat zehazten du alde guztietan edo baten batean. 
        
        .edukiontzia {
            display: flex;
            gap: 25px;
          
            /* Zutabea eta lerroa zehaztuz */
            row-gap: 25px;
            column-gap: 75px;
            
            /* Modu laburrean gauza bera */
            gap: 25px 75px; 
        }

Grid

Diseinu sistema bidimentsionala da, "laukietan " oinarritua, hau da, zutabeekin eta errenkadekin erabiltzen da. Laukiko elementuen jatorri-ordenak ez du axola, eta, beraz, oso erraza da Media Query-ekin berrantolatzea.

  • Grid-ekin oso erraza da, honako propietate hauek erabiliko dituzu "item" bat edo batzuk "edukiontzia" elementu baten barruan zentratzeko. 
        
        .edukiontzia {
            display: grid;
            place-items: center;
        }
  • Gridek laukia 2 dimentsiotan aldatzeko aukera eskaintzen dizu. Kasu honetan, zutabeak, ardatz bertikala, zure gustura doitzea bilatzen baduzu, honako propietate hauek dituzu. 
        
        .edukiontzia {
            display: grid;
            grid-template-columns: 100px 2fr 1fr;
        }
  • Ardatz horizontalaren dimentsioa lantzea falta zaigu, errenkadak. Horretarako, honako propietate hauek dituzu. 
        
        .edukiontzia {
            display: grid;
            grid-template-rows: 1fr 2fr;
        }
  • Ez duzu hain lauki aspergarria egin behar, dinamismo pixka bat emango diozu. Banakako gelaxka bakoitzean balioak ezar ditzakezu, baita gelaxkak beste batzuekin gainjarrita ere; hori propietateekin lor dezakezu: 
        
        .edukiontzia {
            display: grid;
        }
        
        .first {
            grid-column-start: 1;
            grid-column-end: 3;
            grid-row-start: 1;
            grid-row-end: 3;
          
            /* Laburki esanda lorpen bera */
            grid-column: 1 / 3;
            grid-row: 1 / 3;
        }
        
        .last {
            grid-column-start: 1;
            grid-column-end: 4;
            grid-row-start: 4;
            grid-row-end: 4;
          
            /* Laburki esanda lorpen bera */
            grid-column: 1 / 4;
            grid-row: 4 / 4;
        }

    Kasu horretan, ilarak eta zutabeak modu banatuan errepikatzea bilatuko duzu, eta, horregatik, propietate horiek erabiliko dituzu.

        
        .edukiontzia {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: repeat(4, 1fr);
        }

Diseinu moldagarria

Diseinu moldagarria web-diseinuaren ikuspegi bat da, webgunearen edukia hainbat gailuren pantaila- eta leiho-tamaina desberdinetara egokitzen duena.

Adibidez, zure edukia mahaigaineko pantailetako zutabe desberdinetan bereizita egon liteke, diseinu hori egokitzeko bezain zabalak direlako.

Edukia hainbat zutabetan banatzen baduzu gailu mugikor batean, zaila izango da erabiltzaileek irakurtzea eta harekin elkarreragitea.

Diseinu moldagarria aukera ematen du zure edukitik eta diseinutik bereizitako diseinu ugari eskaintzeko gailu desberdinei, pantailaren tamainaren arabera.
