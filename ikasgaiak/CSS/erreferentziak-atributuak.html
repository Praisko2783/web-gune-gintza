ID atributua

Aurreko arauek hautatzaileak adierazitako elementuei eragiten diete. HTML elementu bat hautatzeko, haren mota kontuan hartu gabe, id atributua erabil dezakegu. Atributu hori izen bat da, elementuaren identifikatzaile esklusibo bat, eta, beraz, dokumentuaren barruan elementu jakin bat aurkitzeko erabil dezakegu. id atributua erabiliz elementu bat erreferentziatzeko, hautagailuak atributuaren balioa jaso behar du, aurretik zenbakizko karakterea, edo hainbatek deitzen dion bezala burukotxoa, duela (#)
