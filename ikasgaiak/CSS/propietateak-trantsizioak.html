CSS propietateak (trantsizioak)

hoverhover sasiklasearekin eraldaketa dinamikoak egin ditzakegu. Hala ere, benetako animazioak prozesuaren bi urratsen arteko trantsizioa eskatzen du. Horretarako, CSSk honako propietate hauek eskaintzen ditu.

  • transition-property —Propietate honek trantsizioan parte hartzen duten propietateak zehazten ditu. Propietateen izenez gain, all balioa esleitu dezakegu propietate guztiek trantsizioan parte hartuko dutela adierazteko.
  • transition-duration —Propietate honek trantsizioaren iraupena segundotan zehazten du.
  • transition-timing-function —Propietate honek trantsiziorako balioak kalkulatzeko erabiltzen den funtzioa zehazten du. Honako balio hauek daude eskuragarri: ease, ease-in, easeout, ease-in-out, linear, step-start eta step-end .
  • transition-delay —Propietate honek zehazten du nabigatzaileak zenbat denbora itxarongo duen animazioa hasi aurretik.
  • transition —Propietate honek trantsizioaren balio guztiak aldi berean deklaratzeko aukera ematen digu.

Propietate horiek inplementatuz, nabigatzaileari adierazten diogu animazioaren urrats guztiak sortu behar dituela eta elementuaren egungo egoeraren eta propietateek zehaztutakoaren arteko trantsizioa sortu behar duela. Hurrengo arauek aurreko adibidean sartutako eraldaketa animatzeko propietate transition inplementatzen dute.

        
        header {
        margin: 30px;
        padding: 15px;
        text-align: center;
        border: 1px solid;
        transition: transform 1s ease-in-out 0s;
        }
        header: hover {
        transform: rotate (5deg);
        }
        #izenburua {
        font: bold 26px Verdana, sans-serif;
        }

Transition propietateak lau parametro ere jaso ditzake, espazio batez bereizita.

  • Lehen balioa (gure adibidean transform propietateatrantsizioa sortzeko kontuan hartuko den propietatea da.
  • Bigarren parametroa (1s = segundo bat) animazioaren iraupena zehazteko erabiltzen da
  • Hirugarren parametroa (Bézier kurba) trantsizioa nola egingo den zehazten duen balio bat da.
  • Laugaren parametroa (0s = zero segundoa) animazioak hasteko zenbat segundo behar dituen zehazten du.

Trantsizioak propietate bat baino gehiagoren balioak kontuan hartu behar baditu, koma bidez bereizitako propietateen izenak adierazi behar ditugu. Animazioa sortzeko aldatzen diren propietate guztiak kontuan hartu behar direnean, all balioa erabil dezakegu horren ordez.
