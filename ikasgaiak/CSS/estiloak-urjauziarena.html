Urjauziaren kontzeptua

Atarikoa

CSS ur-jauziaz ari garenean, nabigatzaileak HTML elementu bati CSS estiloak aplikatzeko duen algoritmoaz ari gara (arauen eta arauen multzoa). Ez da gehienek uste duten bezain erraza, kasu askotan (asko eta asko!) anbiguotasunak egon baitaitezke, eta nabigatzaileak ebatzi egin behar ditu, lehendabizi zein estilo aplikatu behar duen eta ondoren zein aplikatu behar duen jakiteko.
Imajinatu bi .css fitxategi ditugula lotuta < link > etiketa baten bidez, eta bakoitzean .aipamena hautagailu bera dugula, kolore desberdinak aplikatuta. Bietako zein aplikatuko da azkenean? Agian uste duzu definitu den azkena. Batzuetan hori egia da, baina beste batzuetan ez.
