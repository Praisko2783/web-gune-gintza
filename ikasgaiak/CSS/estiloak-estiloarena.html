Estiloa nola ezarri web-orrietan

Atarikoa

Aurreko kapituluan, HTML dokumentu bat sortzen, haren egitura antolatzen eta edukia irudikatzeko zein elementu diren egokienak zehazten ikasi dugu. Informazio horrek dokumentua definitzeko aukera ematen digu, baina ez du zehazten pantailan nola agertuko den. HTML5 sartu zenetik, zeregin hori CSSren erantzukizuna da. CSS lengoaiak HTML elementuei estiloak esleitzeko jarraibideak ematen ditu, hala nola koloreak, letra motak, tamainak, etab. Estiloak CSS bidez definitu behar dira, eta gero elementuei esleitu, gure orrialderako nahi dugun diseinu bisuala lortu arte.
